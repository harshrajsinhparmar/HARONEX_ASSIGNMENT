# HARONEX_ASSIGNMENT
A full-stack NGO Event Management System built with React, Node.js, and MongoDB that allows NGOs to organize, display, and manage events efficiently.

Features:
  Event Management: Add, view, and delete events with detailed descriptions.
  NGO Management: Manage NGO profiles and their associated events.
  Admin Panel: Secure access for administrators to manage content.
  Delete Controls: Confirmation-based deletion for NGOs and events.
  Responsive Design: Mobile-friendly navbar and adaptive layout.
  REST API: Integrated backend with structured data flow.


Tech Stack:
  Frontend: React (Vite), Tailwind CSS, React Router, Axios
  Backend: Node.js, Express.js, MongoDB (Mongoose)


Future Improvements:
  User authentication with JWT
  Event registration and participation tracking
  Email notifications
  Dashboard analytics for NGOs
